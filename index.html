Escáner Inteligente de Red

NetScan es una aplicación web diseñada para monitorear redes locales y detectar dispositivos conectados en tiempo real. Utiliza un radar visual interactivo que permite identificar direcciones IP, dispositivos activos y actividad dentro de una red WiFi de forma gráfica y fácil de entender.

Este proyecto simula una herramienta de ciberseguridad que ayuda a los usuarios a visualizar los equipos conectados a su red y comprender cómo funciona el monitoreo de redes.
